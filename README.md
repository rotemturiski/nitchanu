# מחשבון מילואים — ניטחנו

מחשבון אינטראקטיבי לחישוב תגמולים, מענקים וזכויות למשרתי ומשרתות מילואים.

## תכונות
- תמיכה בשנים 2023–2026
- חישוב לפי מדרג פעילות (א׳+/א׳/ב׳/ג׳-ה׳)
- מצב משפחתי: ילדים, בן/בת זוג, מפקד, סטודנט
- העלאת PDF — צו 8 / טופס 3010 מאתר המילואים, חילוץ אוטומטי של תאריכים וימים
- מעקב אחרי הגשות — כולל זכאויות שלא מומשו
- שמירה אוטומטית בדפדפן (localStorage)
- שיתוף חישוב בלינק או כתמונה

## הרצה מקומית
```bash
# דרך שרת מקומי (נדרש ל-fetch של policy.json)
npx serve .
```

## דפלוי ל-Vercel
```bash
npm i -g vercel
vercel --prod
```

## עדכון מדיניות שנתית
כל ינואר כשיוצאת מדיניות חדשה — עדכן את `policy.json`.
השדות שמשתנים בדרך כלל:
- `fMax` — תקרת ארנק פייטר
- `expD` — מענק הוצאות אישיות יומי לפי מדרג
- `specT` / `specD` — סף ושיעור תגמול מיוחד
- `vT` / `vAmt` — שובר נופש

גיבוי: `POLICY_FALLBACK` ב-index.html משמש כ-fallback אם fetch נכשל (למשל פתיחה מ-file://).
